# Laravel Admin Starter Template

A **full-featured Laravel Admin Starter Template** with authentication, roles, permissions, and reusable components — ready to clone and use in any new Laravel project.  

This starter template helps you **kickstart admin dashboards** quickly without setting up authentication, RBAC, or basic components from scratch.

---

## Features

- ✅ User authentication (login, registration, password reset)
- ✅ Admin dashboard starter layout
- ✅ Roles and permissions system (RBAC)
- ✅ Reusable components for forms, tables, and navigation
- ✅ Fully seeded test data (users, roles, permissions)
- ✅ Ready to clone and start new projects instantly
- ✅ Uses Laravel best practices
